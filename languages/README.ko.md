## 구글 지도 스크래핑이란?

**구글 지도 스크래핑**은 구글 지도 검색 결과에 나오는 업체 정보를 자동으로 수집하는 방법입니다. 상호명, 전화번호, 주소를 하나하나 복사-붙여넣기 하는 대신, 스크래퍼가 모든 작업을 대신 처리해줍니다. 몇 시간 걸리던 일이 몇 분이면 끝납니다.

구글 지도에는 수백만 개의 업체가 등록되어 있고, 연락처, 고객 리뷰, 영업시간, 웹사이트 URL, GPS 좌표 같은 데이터가 공개되어 있습니다. 영업팀, 마케팅 대행사, 창업자에게 이건 잠재 고객의 보물창고입니다.

### 왜 기업들이 구글 지도 데이터를 필요로 할까?

구글 지도 기반 **영업 리스트 자동 수집**에 대한 수요가 폭발적으로 늘고 있습니다. 이유는 간단합니다:

- **영업 & 신규 개척**: 영업팀이 업종, 지역, 별점으로 세분화된 타겟 리스트를 만듭니다. 강남에서 별점 4.5 이상, 리뷰 100개 이상인 카페라면 잘 되는 가게일 확률이 높고, B2B 서비스 예산도 있을 겁니다.

- **시장 조사**: 지역별 경쟁사 밀도, 가격대 패턴, 고객 반응을 파악하면 데이터 기반의 사업 확장 결정을 내릴 수 있습니다.

- **로컬 SEO**: SEO 대행사가 구글 지도 데이터를 추출해서 지역 검색 경쟁 현황을 분석하고, 클라이언트의 성과를 경쟁사와 비교합니다.

- **부동산**: 중개인과 투자자가 매물 주변의 상권 밀도를 분석해서 입지를 평가합니다.

- **인재 채용**: 헤드헌팅 회사가 성장 중인 업체(리뷰가 빠르게 늘어나는 곳, 최근 소유자 인증된 곳)를 찾아서 영업합니다.

문제는? 수작업 데이터 수집은 끝이 없다는 겁니다. 업체 100곳 정보를 손으로 복사하면 6시간 이상 걸립니다. 구글 지도 스크래퍼를 쓰면 3분이면 됩니다.

## 활용 사례

<p align="center">
  <img src="../assets/use-cases.svg" alt="구글 지도 스크래퍼 업종별 활용 사례" width="700">
</p>

### 1. 영업팀 & 리드 생성 대행사

가장 많이 쓰이는 용도: **B2B 영업 리스트 구축**. SDR(영업개발담당자)이 추출 데이터로 정밀 타겟팅:

- "홍대 카페"로 검색 → 500개 카페의 전화번호와 웹사이트 추출
- 별점 4.0 이상, 리뷰 50개 이상으로 필터 → 안정적인 업체 식별
- CSV 내보내기 → CRM에 가져오기 → 아웃리치 시작

**결과**: 하루 종일 리스트 만드는 대신, 10분이면 검증된 영업 리스트 완성.

### 2. 부동산 중개인

부동산 영업과 투자자가 구글 지도 데이터를 활용:

- 매물 주변 상권 밀도 분석
- 성장 중인 업체 발견 (리뷰 많음 = 장사 잘 됨)
- 지역 사업자 네트워크 구축

### 3. 헤드헌터 & 인재 채용

인재 회사가 잠재 고객을 찾는 방법:

- 신규 오픈 업체 (리뷰 적음, 최근 등록) → 인력 필요할 가능성 높음
- 급성장 업체 (리뷰 급증) → 채용 중일 가능성
- "직원 모집" 표시된 업체 → 즉시 접근

### 4. SEO 대행사

SEO 전문가가 구글 지도 스크래핑을 활용:

- **경쟁 분석**: 클라이언트의 로컬 시장에 경쟁사가 몇 개인지?
- **사이테이션 구축**: NAP(이름, 주소, 전화) 일관성 확인
- **리뷰 모니터링**: 경쟁사의 리뷰 빈도와 감성 추적

### 5. 시장 조사

컨설턴트와 리서처가 데이터를 추출해서:

- 지역별 업종 집중도 매핑
- 지역 간 가격대 패턴 분석
- 미개척 시장 발견 (업체 밀도 낮음 + 인구 많음)

## 사용 방법

### 1단계: 검색하기

[구글 지도](https://www.google.com/maps) 또는 [빙 지도](https://www.bing.com/maps)를 열고 타겟을 검색:

- "강남 맛집"
- "홍대 카페"
- "판교 IT회사"
- "부산 미용실"

### 2단계: 추출하기

확장 프로그램 아이콘을 클릭. **Start**를 누르면:

1. 화면에 보이는 모든 업체 정보를 수집
2. 자동 스크롤로 추가 결과 로드
3. 각 업체의 상세 정보 가져오기
4. 중복 실시간 제거

### 3단계: 내보내기

**CSV** 클릭 — 끝. 파일은 바로 사용 가능:

- **CRM**: HubSpot, Salesforce, Pipedrive
- **스프레드시트**: 구글 시트, 엑셀, Airtable
- **이메일 도구**: Mailchimp, Stibee, SendGrid

## 자주 묻는 질문

### 구글 지도 스크래핑은 합법인가요?

공개된 업체 정보를 수집하는 것은 대부분의 법적 관할권에서 일반적으로 합법으로 간주됩니다. 상호명, 전화번호, 주소는 업체가 스스로 공개한 데이터입니다. 다만 속도 제한을 지키고, 데이터를 스팸에 사용하지 않으며, 개인정보보호법을 준수해야 합니다.

### 한 번에 몇 건까지 수집할 수 있나요?

오픈소스 확장 프로그램으로는 검색당 **최대 약 200건**이 현실적입니다. [Pro 클라우드 버전](https://gmapsscraper.io?ref=github-profile-ko)은 실질적으로 무제한입니다.

### 빙 지도에서도 되나요?

네. **구글 지도와 빙 지도 모두** 지원합니다. 어떤 플랫폼에 있는지 자동으로 감지합니다.

### API 키가 필요한가요?

아니요. 확장 프로그램은 브라우저 안에서 완전히 동작하며, 외부 의존성이 없습니다.

### 어떤 브라우저를 지원하나요?

Chromium 기반 모든 브라우저: Chrome, Edge, Brave, Opera, Vivaldi, Arc.

### 데이터 정확도는?

구글 지도 또는 빙 지도에 표시된 정보를 그대로 추출하므로 소스와 동일한 정확도입니다. 전화번호와 주소의 정확성은 보통 **90% 이상**입니다.
